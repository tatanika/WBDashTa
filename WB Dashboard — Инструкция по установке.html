🛍️ WB Dashboard

Инструкция
по установке

Персональный дашборд для продавцов Wildberries — статистика, склады, стоимость логистики, позиции в поиске и реклама.

✦ Бесплатно · Работает на твоём компьютере · Данные только у тебя
⏱️
Время установки
5–10 минут
💻
Операционная система
Windows 10 / 11
🔑
Нужно иметь
Аккаунт WB
1
Установи Node.js
1
Открой сайт nodejs.org

Перейди на nodejs.org/ru и скачай версию LTS (рекомендуемую).

2
Запусти установщик

Открой скачанный файл .msi, нажимай «Далее» → «Далее» → «Установить». Всё по умолчанию.

3
Проверь установку

Нажми Win + R, введи cmd, нажми Enter. В открывшемся окне введи:

node --version

Должно появиться что-то вроде v22.0.0 — значит всё хорошо.

ℹ️
Node.js устанавливается один раз В следующий раз при обновлении дашборда повторять не нужно.
2
Скачай дашборд
1
Открой страницу на GitHub

Перейди по ссылке:

🔗
Ссылка на дашборд
https://github.com/tatanika/WBDashTa
2
Скачай ZIP-архив

Нажми зелёную кнопку Code → выбери Download ZIP.

3
Распакуй архив

Правой кнопкой на скачанный файл → Извлечь всё. Выбери удобное место, например C:\WBDashboard\.

3
Получи API-токен на Wildberries
⚠️
Токен нужен для доступа к данным твоего магазина Он хранится только на твоём компьютере и никуда не передаётся.
1
Открой кабинет продавца

Зайди на seller.wildberries.ru под своим аккаунтом.

2
Перейди в раздел API

Нажми на своё имя (справа вверху) → Настройки → Доступ к API.

3
Создай новый токен

Нажми «Создать новый токен». Введи любое имя (например: «Мой дашборд»).

4
Выбери разрешения

Отметь галочками следующие разрешения:

Контент
Аналитика
Статистика
Цены и скидки
Склад
Продвижение
5
Скопируй токен

Нажми «Создать». Токен появится один раз — скопируй его полностью (Ctrl+A, затем Ctrl+C).

🔒
Никому не передавай токен! С ним можно читать данные твоего магазина. Если токен утёк — удали его в кабинете WB и создай новый.
4
Запусти дашборд
1
Открой папку с дашбордом

Зайди в папку, куда распаковал архив (например C:\WBDashboard\WBDashTa-main\).

2
Запусти start.bat

Дважды кликни на файл start.bat. Откроется чёрное окно — подожди, оно само установит зависимости при первом запуске.

3
Открой браузер

Когда в чёрном окне появится текст WB Dashboard → http://localhost:3000, открой браузер и перейди по адресу:

http://localhost:3000
5
Введи токен в дашборде
1
Экран первого запуска

При первом открытии ты увидишь экран настройки вместо карточек товаров — это нормально.

2
Вставь токен

Кликни в поле ввода и нажми Ctrl+V — вставь токен, который скопировал на шаге 3.

3
Нажми «Подключить магазин»

Дашборд проверит токен и начнёт загружать данные. Первая загрузка занимает 1–2 минуты.

🎉
Готово! Дашборд настроен. Теперь каждый раз просто запускай start.bat и открывай http://localhost:3000 — всё будет работать автоматически.

Как пользоваться каждый день
▶
Запуск

Двойной клик на start.bat → открыть браузер → http://localhost:3000

🚚
Обновляй ИЛ и ИРП каждый понедельник

Нажми кнопку «🚚 Логистика» в верхней панели и введи актуальные значения из раздела Поставки и заказы → Тарифы в кабинете WB. Дашборд напомнит сам, когда значения устареют.

■
Завершение работы

Просто закрой чёрное окно (или нажми Ctrl+C в нём) и закрой браузер.

?
Частые вопросы
❓ Чёрное окно открылось и сразу закрылось

Убедись, что Node.js установлен. Попробуй запустить start.bat правой кнопкой → «Запустить от имени администратора».

❓ Данные долго загружаются

Первый запуск или нажатие «Обновить» занимает 30–60 секунд — WB API имеет ограничение на скорость запросов. Это нормально.

❓ Токен не принимается (ошибка 401)

Убедись что токен скопирован полностью — он очень длинный (несколько сотен символов). Попробуй создать новый токен в кабинете WB.

❓ Сайт не открывается по адресу localhost:3000

Проверь что чёрное окно запущено и не закрыто. Попробуй другой браузер. Антивирус иногда блокирует локальный сервер — добавь папку в исключения.
