Вставь API-токен из кабинета Wildberries — дашборд подключится к твоему магазину и загрузит все данные автоматически.

1 Открой seller.wildberries.ru
2 Настройки → Доступ к API → Создать новый токен
3 Выбери разрешения: Контент, Аналитика, Статистика, Цены, Склад, Продвижение
4 Скопируй токен и вставь ниже

🔒 Токен сохраняется только на твоём компьютере в файле wb_token.json — никуда не передаётся.
